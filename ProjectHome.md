A small utility to listen to mp3's on any linux OS without any codecs.

You need Flash player for Firefox to use it .

What you need.

1) php cli or apache
2) download the zip file attached
3) extract to a folder say phpMp3
4) copy your mp3's in the same folder.
5) from your command prompt/cli or apache just run
> php createTracks.php
5) open the index.html in firefox and enjoy.
